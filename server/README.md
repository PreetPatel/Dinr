# Running the backend 

## Installation

```
npm install
```

## Running the app

#### Please ensure you have set up the Zomato API key. More details about it can be found [here](https://github.com/PreetPatel/Dinr/wiki/Development-Environment#api-keys)

### development
```
npm run start
```

### watch mode
```
npm run start:dev
```

### production mode
```
npm run start:prod
```

## Test

### unit tests
```
npm run test
```

### e2e tests
```
npm run test:e2e
```

### test coverage
```
npm run test:cov
```
